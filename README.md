# nuViz Editorial Site

Mobile-first editorial website for nuViz, a creative studio offering photography, videography, music videos, and design.

## Stack
- React + Vite + TypeScript
- React Router
- Tailwind CSS

## Getting started
Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

Type-check the project:

```bash
npm run lint
```

Build for production:

```bash
npm run build
```
